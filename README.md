# FAST - Ivo Mendes
This repository contains a work documentation document developed as part of my involvement with the FEUP Academic Solar Team (FAST), where I am a member of the Electrical Systems Department.
The document presents the electrical power and distribution architecture of Cristal, a solar-powered autonomous vessel built to compete in European university competitions. It covers the design decisions, key engineering challenges, and the validation work carried out in a real maritime environment."
